C'est quoi ARDUINO

  1. Definition et origine
    Definition
    Arduino est une plaquete imprimeé avec un microcontroleur programmable en utilisant le language C a partir de son IDE (Environnement de Developpement)
    Origine
    Arduino a été creé en 2005 en INDE par un groupe d'enseignant et d'étudiant

Les differents types de carte ARDUINO

Il existe plusieurs type de carte ARDUINO dont les trois principales sont :

  • ARDUINO MEGA

  • ARDUINO UNO
  • ARDUINO NANO

A quoi sert un ARDUINO

ARDUINO sert a faire de la domotique , de l'automatisation (...).

Il permet de creer : des maisons intelligentes,des voitures autonome et plein d'autre projet.
